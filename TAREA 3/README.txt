<<FORMA DE COMPILAR>>

	Para la compilacion de los archivos .cpp se utilizo el metodo de MinGW con el idle Visual Studio Code.
	Al usar ese idle con presionar F6 compila.


<<EXTRA>>

	El archivo cache-diccionario.cpp al no tener funcion main da un warning al usar 
	g++ cache-diccionario.cpp -Wall -o cache-diccionario
